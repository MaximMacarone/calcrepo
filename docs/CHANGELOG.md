# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-22

### Добавлено
- Базовый функционал калькулятора (сложение, вычитание, умножение, деление)
- SwiftUI интерфейс с современным дизайном
- Unit-тесты для всех операций калькулятора
- CI/CD настройка с GitHub Actions
- Автоматическое тестирование и сборка
- Автоматический merge PR после успешных проверок
- Документация проекта

### Технические детали
- Swift 6.1+
- iOS 17.0+
- Swift Package Manager
- Swift Testing framework
