# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-01-XX

### Добавлено
- Базовая функциональность калькулятора
  - Сложение, вычитание, умножение, деление
  - Очистка калькулятора
  - Отображение результатов
- SwiftUI интерфейс калькулятора
- Unit-тесты для всех операций калькулятора
- CI/CD настройка с GitHub Actions
  - Автоматический запуск тестов
  - Автоматическая сборка приложения
  - Генерация документации
  - Создание релизов
- Документация проекта
- План выполнения лабораторной работы

### Технические детали
- Swift 6.1+
- iOS 17.0+ deployment target
- Swift Package Manager для модульной архитектуры
- Swift Testing framework для unit-тестов
